# SA-BNet 探索性实验报告

> 实验日期：2026-04-14  
> 模型：DeepLabV3+ MobileNetV2 + Spatial-CBAM + SAFF + Boundary Aux  
> 目的：验证 batch_size 从 16 → 32 后的收敛行为，并为后续超参优化提供基线参考

## 4. 主要观察

### 4.1 batch_size=32 可行性 ✅
- 5 epoch 内 Loss 从 0.66 → 0.14，mIoU 从 56.7% → 71.4%，收敛趋势健康。
- 显存占用约 14 GB，远未触及 32 GB 上限，有余量继续增大。

### 4.2 Abuth 类别不稳定 ⚠️
- Abuth IoU 在 Epoch 3→4 出现回落（45% → 28%），Epoch 5 回升至 41%。
- cotton→abuth 混淆率在不同 epoch 差异较大（2%~22%）。
- **根因**：Abuth 样本较少、类间相似度高，5 epoch 时模型尚未充分区分细粒度特征。

### 4.3 学习率已正确缩放
- 将 lr 从 0.01 → 0.02（线性缩放 ×2）后，loss 曲线无震荡，梯度稳定。
